Allow Item Uploads

When enabled, users can upload new items to the marketplace. Disable this during database backups or data migrations.

When enabled, users can initiate trades and create orders. Disable this to prevent new transactions during maintenance.

When enabled, users can browse and search for items. Disable this for read-only maintenance access.
