## FAQ

**Q: How many friends can I refer?**
A: Unlimited! No cap on referrals.

**Q: When do I get paid?**
A: Credits are added instantly:
- Signup bonus: immediately
- Item approval bonus: when admin approves
- Purchase bonus: when they checkout

**Q: Can I use a referral code?**
A: Yes! If a friend referred you, use their code at signup.

**Q: What if I didn't use a code at signup?**
A: You can't retroactively apply it. Use it for future accounts.

**Q: Can one friend earn from multiple codes?**
A: No, each account uses only one referral code.

**Q: What's the referral code format?**
A: REF followed by numbers and letters (e.g., REF12ABC)

**Q: Do the bonuses expire?**
A: No, they're permanent credits in your account.

**Q: Is there a limit to earnings?**
A: No limit! Earn as much as you can.
